About α / β Logging

Alpha (α) = acute angle between the core axis and the structure plane (0–90°).
Beta (β) = angle measured clockwise (looking down-hole) from the bottom-of-hole reference line to the lower point of the structure ellipse (0–360°).

This is the standard workflow for oriented core logging in commonwealth exploration drilling (AusIMM / JORC practice). The result can be added directly to the active station as a Fault Core measurement.
